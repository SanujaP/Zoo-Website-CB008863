The Elephant Show

Dehiwala National Zoological Gardens presents The Elephant Show, an entertaining show presented by well trained Elephants. These animals would be doing extreme acts which would amaze all the visitors.
Please note that there will be extra charges for this show, kindly refer to the on location ticket counter for purchasings.


Fee Per Person - LKR 5000.00
Start Time - 4.30 PM
Show Days - Every work day
